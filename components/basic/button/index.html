Button 按钮

Taffy ui包括的按钮有主按钮、次按钮、虚线按钮、文本按钮、链接按钮、block按钮 

			<button class="button primary">Primary Button</button>
			<button class="button second">Second Button</button>
			<button class="button block">Block Button</button>
			<button class="button dashed">Dashed Button</button>
			<button class="button text-button">Text Button</button>
			<button class="button link">Link Button</button>

Disabled 禁用状态

Taffy ui还包括以上这些按钮的disabled状态 

			<button class="button primary disabled">Primary Button</button>
			<button class="button second disabled">Second Button</button>
			<button class="button block disabled">Block Button</button>
			<button class="button dashed disabled">Dashed Button</button>
			<button class="button text-button disabled">Text Button</button>
			<button class="button link disabled">Link Button</button>
