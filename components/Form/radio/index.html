Radio 单选框

在一组备选项中进行单选

基础用法

由于选项默认可见，不宜过多，若选项过多，建议使用 Select 选择器。 


      
        <input type="radio" class="radio" id="option1" name="radioGroup" value="option1">
        <label for="option1">选项1</label><br>
        <input type="radio" class="radio" id="option2" name="radioGroup" value="option2">
        <label for="option2">选项2</label><br>
        <input type="radio" class="radio" id="option3" name="radioGroup" value="option3">
        <label for="option3">选项3</label>

禁用状态

单选框不可用的状态。 


      
        <input type="radio" class="radio" id="option4" name="radioGroup" value="option4" disabled>
        <label for="option4">选项1</label><br>
        <input type="radio" class="radio" id="option5" name="radioGroup" value="option5" disabled>
        <label for="option5">选项2</label><br>
        <input type="radio" class="radio" id="option6" name="radioGroup" value="option6" disabled>
        <label for="option6">选项3</label>
